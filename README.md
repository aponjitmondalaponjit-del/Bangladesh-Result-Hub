# 🇧🇩 Bangladesh Result Hub (BRH)

Bangladesh Result Hub (BRH) is a Telegram Bot project for checking official examination results in Bangladesh.

## Version 1 Features

- SSC Result
- HSC Result
- Dakhil Result
- Alim Result
- All Education Boards
- User Profile
- Referral System
- Rewards System
- Auto Result Notification
- Notice
- Support
- Admin Panel
